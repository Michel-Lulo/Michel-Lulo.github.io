How to Navigate NRG Systems Through Google Colab

Hello. This is a simple web page that will guide you through using python in order to quickly export site data from NRG Systems.

Here is a link to the Official Google Colab that we will be using.

Make a copy

To make a copy you must have a google drive account. After that, making a copy is as simple as going to file and clicking on "Save a copy in Drive".

Set a date range

Begin by setting a start and end date in the highlighted section under "Site Info". Ensure that the format you set each of the dates is "year-month-day".

Choose a project site

After clicking the run all command, a list of projects should appear under "Site Info" as seen in the image below: "

Simply keep in mind the "siteNumber" and "siteDescription" for later use.

Rewrite the program

Under "Execute", rewrite the site number at "insert site number" and the site name at "insert site name". Make sure to delete everything in the parenthesis and the parenthesis itself.

Re-run the program

Once again, click the "run-all" command.

Determine which channel/s to graph

A list with a short overview of every channel should now appear under "Execute".

Decide and keep in mind which channels you want to analyze, for later on.

How to graph the data
